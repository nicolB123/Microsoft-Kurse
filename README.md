# Microsoft-Kurse
 Die wichtigsten Inhalte der Microsoft Learn Kurse dargestellt.
 Im Power BI Kurs führen die Labs linear zu einem finalen Ergebnis deshalb der Fokus auf dem Inhaltsverzeichnis (Topics) und Endresultat, wohingegen die Labs im Fabric Kurs unterschiedliche Dinge betracheten und damit die Lab Inhalte direkt im Ordner sind
