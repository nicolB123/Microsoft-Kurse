# Microsoft-Kurse
 Die wichtigsten Inhalte der Microsoft Learn Kurse dargestellt.
 Im Power BI Kurs führen die Labs linear zu einem finalen Ergebnis deshalb liegt der Fokus auf dem Inhaltsverzeichnis (Topics bzw. README) und dem Endresultat, wohingegen die Labs im Fabric Kurs unterschiedliche Dinge betrachten und damit die Lab Inhalte direkt im Ordner sind.

Azure Databricks: Ich habe den Kurs "Course DP-3011-A: Implement a Data Analytics Solution with Azure Databricks" in der Theorie und Module Assessments abgeschlossen. Leider lässt die Hochschule aufgrund ihrer Azure Security Policy "private" Databricks Workspaces nicht zu, weshalb die praktischen Labs nicht bearbeitet werden konnten. Ein grundlegendes Verständnis in Theorie und Struktur ist vorhanden (In weiten Teilen hohe Ähnlichkeit zu MS Fabric).
